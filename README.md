# General Assembly: Project 4: CupidREACT.
**Cupid is an online dating site. Similar to Tinder/Bumble** 

**Cupid functions**
- Register
- Login
- Update profile
- Add/update interests
- Like/Unlike possible matches
- View current matches
- Unmatch a match
- Delete account
